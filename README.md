## Overview
The Organization Dashboard is a responsive web-based interface designed for environmental authorities and municipal bodies. It serves as a command center to view, track, and resolve pollution reports submitted by citizens in real-time.

## Features âœ¨
* **Centralized Report Viewer:** Tabular display of all incoming pollution reports with severity levels and categories.
* **Interactive Map Visualization:** View report locations on a map to identify pollution hotspots and prioritize responses.
* **Status Management:** Update report status from "Pending" to "Resolved" and upload "Proof of Resolution" images.
* **Real-Time Sync:** Instant updates from the mobile app ensuring immediate visibility of new incidents.
